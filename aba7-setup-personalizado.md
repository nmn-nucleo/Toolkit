## SECAO 2 — Como funciona o quiz (preview)

6 perguntas. ~5 minutos. Cada pergunta tem porque ela importa pro resultado.

### Pergunta 1: Qual seu nivel de experiencia?
**Opcoes:** "Nunca programei" / "Programo basico (HTML/CSS, scripts simples)" / "Programo profissionalmente" / "Sou dev senior, quero otimizar workflow"

**Pra que serve:** define a linguagem do manual, a profundidade dos exemplos, e que tipo de skill recomendar (skills basicas vs avancadas).

### Pergunta 2: Qual sua area de atuacao principal?
**Opcoes:** "Designer / criativo" / "Analista de dados" / "Programador" / "Empresario / gestor" / "Advogado / contador / consultor" / "Marketing / conteudo" / "Educador / pesquisador" / "Estudante / curioso" / "Outro" (campo livre)

**Pra que serve:** define skills relevantes (analise de planilha pra dados, design system pra criativos, automacao de docs pra juridico), exemplos no manual, e ideias de agentes especificas pra essa area.

### Pergunta 3: Qual seu sistema operacional?
**Opcoes:** "Windows" / "macOS" / "Linux"

**Pra que serve:** ajusta os comandos de instalacao, paths, e scripts (PowerShell vs Bash vs zsh).

### Pergunta 4: O que voce ja tem instalado?
**Multiplas:** "Node.js" / "Python" / "Git" / "Conta GitHub" / "Nenhum disso" / "Nao sei"

**Pra que serve:** se voce nao tem nada, o setup script comeca instalando os pre-requisitos. Se ja tem tudo, pula direto pro agente.

### Pergunta 5: Claude Code, Codex, ou recomendacao?
**Opcoes:** "Quero Claude Code" / "Quero Codex" / "Recomende pra mim baseado no perfil"

**Pra que serve:** se voce escolhe um, o setup foca nele. Se pede recomendacao, o sistema decide baseado nas perguntas anteriores (iniciante + Windows + sem nada instalado = Codex, por barreira menor; dev senior + customizacao = Claude Code).

### Pergunta 6: Qual seu objetivo principal nos proximos 3 meses?
**Opcoes:** "Construir um produto/site" / "Automatizar tarefas chatas do trabalho" / "Analisar dados / criar dashboards" / "Aprender programacao na pratica" / "Acelerar entregas em projetos pagos" / "Outro" (campo livre)

**Pra que serve:** define os 3 projetos sugeridos no manual e os agentes recomendados. **Esta e a pergunta mais importante** — define o vetor de personalizacao.

### Bonus opcional: tema/universo de referencia
**Campo livre.** Exemplos: "espacial / NASA", "natureza", "minimalismo japones", "cyberpunk", "futebol", "literatura russa".

**Pra que serve:** se preenchido, o manual e o tema visual sao adaptados (Rosa pediu espacial, ganhou "Do zero a orbita, em um comando de cada vez"). Se vazio, usa tema padrao Get It Easy.

# Notas pro Claude Code (instrucoes de implementacao)

Esta e a aba mais complexa de implementar. Sugestoes:

### Quiz interativo (Secao 4)
- Implementar em vanilla JS (sem framework, mantendo single-file)
- Estado: objeto JS global que armazena respostas
- Navegacao: setas do teclado + botoes "Anterior/Proximo"
- Progress bar: visual no topo (1/9, 2/9, etc.)
- Validacao: nao permite avancar sem responder
- Persistencia: localStorage pra continuar de onde parou (caso recarregue pagina)
- Resultado: gerado client-side a partir das respostas (sem backend)

### Geracao de artefatos (depois do quiz)
Cada artefato deve ser gerado dinamicamente via JS:

**Prompt personalizado:** template literal com substituicoes baseadas em respostas
```js
function gerarPrompt(respostas) {
  return `Sou ${respostas.area}, ${respostas.nivel}. 
  Quero usar ${respostas.agente} pra ${respostas.objetivo}.
  ...`;
}
```

**CLAUDE.md / AGENTS.md:** template baseado em area + nivel + objetivo
**Skills recomendadas:** lookup table de area → skills relevantes
**3 projetos:** lookup table de area + objetivo → projetos sugeridos
**3 agentes:** lookup table de area → agentes sugeridos
**Comandos setup:** template baseado em SO + agente + ja instalado

### Botoes de copy/download
- Cada artefato com botao "Copiar pra clipboard" + botao "Baixar como arquivo"
- Use `navigator.clipboard.writeText()` pra copy
- Use `Blob` + `URL.createObjectURL` pra download

### Tela de resultado
- Layout: hero com slogan personalizado + grid de 6 cards de artefatos
- Cada card expansivel pra mostrar preview do conteudo
- Sticky CTA no rodape: "Quero o pacote completo (R$ 197)"

### Integracao premium (futura)
- Por enquanto, o premium pode ser CTA pra formulario de contato (email pra comprar)
- Quando estiver pronto: integracao com gateway de pagamento brasileiro (Hotmart, Eduzz, Stripe)
- Pos-compra: redireciona pra pagina com download de PDF + ZIP + script + acesso Discord

### Cases reais (Secao 3)
- Cards visuais com mockups (pode ser screenshots de manuais reais ou ilustracoes)
- Cada card clicavel abre modal com preview maior
- "Espalhar" cases ao longo da pagina pra demonstrar valor sem virar feira de produtos

### Tabela de tiers (Secao 5)
- Tabela responsiva (desktop) ou cards empilhados (mobile)
- Tier "Comunidade" como recomendado (badge "MAIS POPULAR" ou similar)
- Botoes de CTA com cores diferentes pra cada tier

### Performance
- Quiz roda 100% client-side
- Templates dos artefatos podem ser carregados sob demanda (1 modulo JS por area)
- Total: aba nao deve passar de +50KB no index.html

### Acessibilidade
- Quiz navegavel 100% por teclado
- Screen reader friendly (ARIA labels em cada step)
- Foco automatico no primeiro elemento de cada step
- Possibilidade de voltar e mudar resposta
